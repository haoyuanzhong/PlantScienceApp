# 生活常见植物科普 App

## 项目简介
本应用是一款面向日常生活的植物科普Android应用，旨在帮助用户认识生活中常见植物，并了解其基本养护知识。应用通过图文展示的方式，帮助用户在日常环境中识别植物，同时提供简单实用的养护建议。

## MVP 核心功能
- 室内/户外植物分类与列表展示
- 详尽的植物图文科普与养护指南（光照、浇水、生长环境等）
- 植物名称快捷搜索
- 个人本地收藏夹（收藏/取消收藏植物）

## 技术栈
- 开发工具：Android Studio
- 开发语言：Kotlin
- 核心UI组件：RecyclerView, CardView
- 本地存储：SharedPreferences（收藏功能）
- 图片加载：Glide

## 页面结构
1. **首页**：应用主入口，展示室内/户外植物分类卡片和收藏按钮
2. **植物列表页**：展示对应分类下的植物卡片，支持按名称搜索
3. **植物详情页**：展示植物完整科普信息和专业养护建议
4. **收藏页**：统一管理用户已收藏的植物，支持取消收藏
